# Farm Digital Twin - Mobile Collector Client

This is the Flutter mobile data collection app for mapping plantation assets.

## 🚀 Running the App

To run the Flutter app on Chrome/Web:

```bash
# Navigate to the mobile directory
cd mobile

# Fetch the Leaflet mapping dependencies
flutter pub get

# Run the app in Chrome
flutter run -d chrome
```

For more details on running the backend database and APIs, please refer to the [Main Project README](../README.md).
